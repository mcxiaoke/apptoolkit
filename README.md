App Toolkit
===========================
<pre>
App Tookit 是一个简洁的应用管理工具，可以备份应用的安装文件和数据，支持批量备份。
</pre>
当前版本的功能：
<pre>
注意：默认备份目录为/sdcard/AppToolkit
</pre>
<pre>
*  备份某个应用的安装文件到SD卡
*  备份某个应用的数据到SD卡 (需要ROOT权限)
*  批量备份已安装应用的安装包到SD卡 (需要ROOT权限)
*  批量备份已安装应用的数据到SD卡 (需要ROOT权限)
*  查看应用的详细信息
*  卸载已安装的应用
</pre>

<pre>
App Tookit is a open source Android app for backup app apk and data.
it is under Apache License 2.0.
</pre>
current version include following features:
<pre>
*  Backup app's apk file to sdcard
*  Backup app's data to sdcard (need root)
*  Batch backup apks to sdcard (need root)
*  Batch backup data to sdcard (need root)
*  View installed app's detail
*  Uninstall installed app
</pre>
